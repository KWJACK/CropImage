프로그램 사용 안내서

이 프로그램은 이미지 파일(jpg, png, bmp)를 읽어서 자신이 원하는 영역을 잘라내는데 사용할 수 있습니다.
저장 파일 형식은 bmp 입니다. 또한 이진화 기능을 제공합니다

 이진화된 영상은  images 폴더에 저장됩니다.
잘라낸 이미지의 저장경로와 이진화 결과영상 저장경로를 프로그램 내에서 바꿀 수도 있고,
(별도의 파일)을 통해서 바꿀 수 있습니다.

 사용시 파일 생성 규칙 ad0000_이 있으므로 규칙에 맞도록 사용해야합니다.
"ad0000_" 만 입력하면 저장되지 않고 "ad0000_테스트_0"으로 문자열, 인덱스까지 작성해야 저장됩니다
인덱스는 자동으로 카운팅되니 각 문자열 최초 1회에 한해서 0만 적어주면 빠르게 이용가능합니다.


 만약 이 기능을 원하지 않으시다면 CBMPZoomView::OnSaveCropImageFile() 함수
조건문if(dlg.DoMal() == IDOK) 내를 주석처리하면 됩니다. 또한, m_IDOK는 항상 TRUE로 바꿔주세요

인덱스는 0~9까지만 가능합니다.
따라서 글자 길이가 10인 글자는 분할해서 저장해야 합니다.
분할 예시) (5,5), (3,7), (4,6) etc..

메뉴 설명
[메뉴]-[이진화]-[1bpp 파레트 반전] : 1bpp 파일에 대해서 파레트정보, 영상정보를 비트 반전 시킵니다
[메뉴]-[이진화]-[24bmp 이진화] : 잘라낸 이미지들이 있는 폴더 전체 영상을 1bpp 파일로 변환합니다.
[메뉴]-[설정]-[자른 이미지 저장하기] : 이미지 저장은 자동으로 동작되지만, 수동으로도 동작가능합니다
[메뉴]-[설정]-[이미지 저장경로 설정하기] : 잘라낸 파일이 저장된 경로를 지정합니다.(Default: 어디)
[메뉴]-[설정]-[이미지 결과경로 설정하기] : 24bmp 사진을 1bpp 파일로 변경시 저장할 path를 설정합니다
결과경로는 가급적 이미지 저장경로와 다른 곳으로 설정해주세요

팁
- 스크롤바를 움직이지 않는게 좋습니다.
- 왼쪽 제일 하단 이진화 영상 미리보기 창은 실제 크기로 나타내는 것이 아니라,
 미리보기 창 해상도에 따라 출력 사이즈가 다르니 실제 크기와 다를 수 있습니다
- 가급적 프로그램은 전체화면에서 사용해주시길 바랍니다.
